### 1. code&data
Core code and data directory.

#### 1.1 data
- **data_tif:**   Stores raw remote sensing data in `.tif` format.
- **data_csv:**  Stores raw data in `.csv` format.

#### 1.2 data_process
Data preprocessing and correction.

- **BRDF_correction**
  - `BRDF_functions.py`  Core functions for BRDF correction.
  - `BRDF_functions_process.py`  Script to run BRDF correction.

- `sif_filter.py`   Filters and handles anomalies in SIF data.

- `tif_clean_avg.ipynb`   Removes anomalies in tif data and computes average values.

- `tif_to_csv.ipynb`  Merges processed tif data and exports them to csv format.

#### 1.3 draw_fig
Figure generation and visualization.

- `fig1_spatial.ipynb`  
- `fig2_spatial.ipynb`  
- `fig3_spatial.ipynb`   Generate spatial distribution maps.

- `fig1_violin.py`  
- `fig2_violin.py`  
- `fig3_violin.py`    Generate violin plots.

- `fig4.py`   Generates additional statistical figures.

- **fig/**  Stores output figures generated by scripts.
