## GRSL_github

GRSL_github
├── code&data
│   ├── data
│   │   ├── data_tif ─ Stores raw data in tif format
│   │   └── data_csv ─ Stores raw data in csv format
│   ├── data_process
│   │   ├── BRDF_correction
│   │   │   ├── BRDF_functions.py ─ Main code for BRDF correction
│   │   │   └── BRDF_functions_process.py ─ Code to run BRDF correction
│   │   ├── sif_filter.py ─ Handles anomalies in SIF data
│   │   ├── tif_clean_avg.ipynb ─ Removes anomalies in tif data and calculates the average
│   │   └── tif_to_csv.ipynb ─ Merges and exports processed tif data to csv format
│   └── draw_fig
│       ├── fig1_spatial.ipynb
│       ├── fig2_spatial.ipynb
│       ├── fig3_spatial.ipynb
│       ├── fig1_violin.py
│       ├── fig2_violin.py
│       ├── fig3_violin.py
│       ├── fig4.py
│       └── fig ─ Output images from fig codes
├── shp: https://figshare.com/articles/dataset/The_data_for_the_NEE_paper_Structural_complexity_biases_vegetation_greenness_measures/23677407/1
│   ├── Amazon_shp ─ Polygon shp files for the Amazon region
│   ├── Cornbelt_shp ─ Polygon shp files for the Corn Belt region
│   └── Amazon_shp_line ─ Line shp files for the Amazon region
└── final-fig ─ Final merged image files
